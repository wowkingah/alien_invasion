# Alien Invasion
外星人入侵游戏。

# ENV
- Python 3.9.1
- PyCharm 2022.2.2
- MacOS

# Running
```shell
# 安装 Pygame 依赖库
$ brew install hg sdl sdl_image sdl_ttf

# 游戏中包含声音等较高级功能的依赖库
$ brew install sdl_mixer portmidi

# 依赖
$ cd alien_invasion
$ pip3 install -r requirements.txt

# 运行
$ python alien_invasion.py
```
